<h2 id="executar"> 
  🎲 Rodando o projeto
</h2>

```bash
# Clone este repositório
$ git clone https://github.com/JoaoDessup/Desafio-Impar.git

# Acesse a pasta do projeto no terminal/cmd
$ cd desafio-impar

# Instale o http-server
$ npm install http-server -g

# Execute o servidor
$ http-server
```
